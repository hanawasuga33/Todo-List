# Todo List App

Простое веб-приложение для создания списка задач. Пользователь может добавлять задачи, отмечать их выполненными и удалять ненужные элементы. Данные сохраняются в браузере через localStorage, поэтому список остаётся после перезагрузки страницы.

## Стек

- HTML
- CSS
- JavaScript

## Как запустить

1. Скачайте или склонируйте репозиторий.
2. Откройте файл `index.html` в браузере.
3. Либо запустите через локальный сервер, например Live Server в VS Code.

## Возможности

- Добавление новой задачи.
- Отметка задачи как выполненной.
- Удаление задачи.
- Сохранение списка в localStorage.
